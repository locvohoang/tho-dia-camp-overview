Measurement Framework
Đo Lường OOH Cộng Hưởng
Direct
QR + Branded Search Lift
Mỗi pano unique QR UTM → track scans trực tiếp. "Nhà tốt quận 12" Google Trends — OOH Aug tốt → search tăng 2-3 tuần sau placement.
Direct
Geo-lift Analysis
So sánh app install / lead từ Q12 postal codes vs districts đối chứng (Q9, Bình Thạnh). Spike = OOH contribution. Cross-validate với digital geo-targeting.
Survey
BHT — Aided/Unaided Recall
Wave 1: W2 Aug (n≥200 Q12). Wave 2: W4 Aug sau OOH heavy. Target: Aided ≥50%, Unaided ≥20%. Nếu wave 1 <30% → optimization trigger ngay.
Proxy
Retargeting Pool Growth
OOH + Social hiệu quả → engagement tăng → retargeting pool lớn hơn. Track pool size W1 Jul → W4 Aug. Target: 3x growth.
Proxy
Reach → Action Funnel
Reach → Impression (3s+) → Engagement → Click → App visit → DwL → Lead. Track % tại mỗi bước. OOH tháng 8 tốt → engagement W1 Aug > Jul W4 baseline.
Proxy
CPR (Cost per Reach)
Budget 8 tuần ÷ (Unique Reach/tháng × 2). So sánh efficiency giữa các OOH options. Input cho GV planning — optimize budget dựa trên Q12 learnings.
⚠️ Trust Signal
Visit → Lead Conversion Rate
Track tỷ lệ app visit → lead submit sau Big Boom. Nếu reach tăng mạnh nhưng visit→lead rate không tăng tương ứng → đây là trust issue, không phải reach issue. Trigger: investigate listing quality (ảnh, giá, pháp lý), agent response rate. Không nên tăng OOH budget — phải fix product/listing trước.

Layer 2 — Niche khu phố
Hyper-local · Differentiation
📍 Đầu hẻm, ngã ba gần agent showcase listings — phân bổ theo agent cluster map

  • OOH + Local Data + Ads listing phải sync cùng khu vực
  • Thiếu L2 = campaign chỉ là awareness đơn thuần, mất differentiation

Layer 3 — Điểm chạm đời thường
Reminder · Brand Recall
📍 Tiệm tạp hóa, tiệm thuốc, trạm xăng, tòa nhà, rạp chiếu phim
  • Activate khi budget OOH còn dư sau L2 + L1
  • Phân bổ proportionally: phường dân đông + listing nhiều → nhiều L3 hơn
  • Indoor boost: +5-8% reach nhóm không commute qua đường lớn
  • Content phù hợp context điểm chạm từng loại địa điểm
